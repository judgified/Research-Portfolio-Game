# Research-Portfolio-Game

Independent research that demonstrates what I have learned throughout the semester taking **THEO280** that can be shared outside of this classroom via a **Python** video game built with **Pygame**.

## Unspoken

This narrative-based game follows **Omar**, who must make a series of difficult choices under social, religious, and legal pressure in a **fictional** setting. There is no single win; outcomes depend on **family honor**, **religious conformity**, and **state suspicion**.

## How to run the game

**No coding background needed.** You are just opening a program and pasting a few lines.

**1. Install Python (one time)**  
Go to [python.org/downloads](https://www.python.org/downloads/) and install it like any other app. On Windows, if the installer asks, turn on **Add Python to PATH**.

**2. Download this project**  
On GitHub: green **Code** → **Download ZIP** → unzip it so you have a folder on your computer.

**3. Open the “command” window**  
- **Mac:** Search for **Terminal** and open it.  
- **Windows:** Search for **Command Prompt** and open it.

**4. Go into your project folder**  
Type `cd` , press the **space bar** once, then **drag your unzipped folder** into the window and press **Enter**.

**5. Paste these two lines, one after the other** (press **Enter** after each; wait for the first to finish):

```text
pip install -r requirements.txt
```

```text
python unspoken.py
```

On some Macs the second line is `python3 unspoken.py` instead.

If a window with the game does not open, ask campus IT or a friend to check that Python is installed—often it is a quick fix.

## Controls

| Action | Input |
|--------|--------|
| Start | Click **Begin story** or press **Enter** |
| Choose | Click a choice (1–3) or press **1**, **2**, or **3** (numpad works) |
| After an ending | **Play again** or **Enter** |
| Quit | **Esc** |

## What's included

- A **start screen** with title, description, and main call-to-action.
- **Five story beats**: Naming, Family, Authority, Violence, and a **variable-based ending** (multiple endings).
- On-screen **stats** (family honor, religious conformity, state suspicion) that change with your choices.
- Some choices **locked** unless your path makes them available.
- **Pygame** UI: story text area, numbered buttons, keyboard shortcuts.
